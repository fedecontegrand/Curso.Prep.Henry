El bucle "for" se utiliza para ejecutar un codigo de manera iterativa, lo que se hace es declarar la variable critica, luego se impone la condicion la cual debe cumplir (es decir, tener un output true) y luego se coloca el comando "++" que hara que la variable incremente su valor en 1 al iterar. Esto hara que se ejecute el codigo en bucle hasta que algun valor de la variable no cumpla la condicion impuesta (que de output false). Se coloca "for" se habre parentesis y se designa la variable colcando comando set, separado de punto y coma redactamos la condicion, y nuevamente separado asi, colcamos el operador incremental "++". Luego abrimos corchetes y dictamos el codigo del bucle.
El comando AND que es "&&" sirve para que el programa ejecute un codigo siempre y cuando se cumplan las dos condiciones impuestas. Si alguna es false, no se ejecutara.
El comando OR que es "| |" sirve para que el programa ejecute un codigo con solo cumplirse una de las condiciones impuestas.
El comando NOT "!" sirve para que nos devuelva el valor booleano opuesto al que nos daria. Es decir que "if (1===1)" nos da un output true, pero al colocar el comando not: "if(!(1===1))" , nos va a dar el opuesto, que es false.
