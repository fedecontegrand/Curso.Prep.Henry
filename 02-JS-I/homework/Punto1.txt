Las variables son elementos que pueden tomar distintos valores segun el tipo de variable, por ejemplo una variable numerica solo tomara el valor que le asignemos a traves de los signos que van del 0 al 9, una variable hexadecimal puede tomar valores que ademas de esos signos, puede usar los simbolos de la a a la f y una variable booleana es una variable binaria que puede tomar dos valores, true o false. Hay diferentes tipos de variables segun el lenguaje. A las variables se les puede asignar nombre y se las puede citar cuando se las necesite.
Una string es una variable cuyo valor se asigna entre " " o ''. Una string sirve para asignar nombres a diferentes tipos de elementos, y se considera estrictamente lo encerrado en las comillas. Se usa para asignar valores textuales.
Las funciones determinan acciones a seguir por quien sea que este corriendo el archivo y obtienen un resultado. Siempre las funciones van seguidas de parentesis, uno de apertura y uno de cierre. Luego entre corchetes se escribe el codigo, que es el que dicta la funcion, y por ultimo asignaremos como "return" a lo que queremos que nos muestre de resultado. La funcion puede ser simple y no requerir de argumentos, en ese caso no se coloca nada entre los parentesis, o puede requerir valores de entrada. Esos valores de entrada se colocan entre los parentesis y hacen al "argumento" de la funcion. Una vez redactada la funcion, podremos ejecutarla dandole los valores de entrada que queremos para ella entre los parentesis y asi obtener el resultado deseado.
Las declaraciones "if" se utilizan para cuando querramos ejecutar un codigo, o una parte de el mismo, en base  a si el valor de una variable cumple una condicion que le imponemos. Entonces existen solo dos escenarios posibles: true o false, y podremos determinar el codigo a correr en caso de que lo cumpla, y otro codigo en el caso de que no.
Los valores booleanos son solo dos: true o false. Son de gran utilidad debido a que nos permiten desarrollar en base al resultado de las variables, es decir, segun el valor que toman ellas.
